A minimalist, fast portfolio built with **React + Vite + Tailwind**. Ready to deploy to Netlify, Vercel, or GitHub Pages.

## Quickstart

```bash
npm install
npm run dev
```

Build for production:
```bash
npm run build
npm run preview
```

## Customize

- Edit content in `src/data.js`
- Update meta tags in `index.html`
- Replace `public/Vikas_Shukla_Resume.pdf` with your latest resume

## Deploy

- **Netlify**: drag & drop `dist/` after `npm run build`
- **Vercel**: `vercel` from project root
- **GitHub Pages**: build and push the `dist/` folder to `gh-pages` branch or use a workflow
